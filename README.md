# 📊 Student Performance Analysis (Excel Project)

This project focuses on analyzing the academic performance of 12th-grade Bio-Maths students using Microsoft Excel.

## 📁 Dataset Structure

The dataset consists of 3 tables:

- **Student Table**
  - Student ID
  - Name
  - Gender
  - Class

- **Subject Table**
  - Subject ID
  - Subject Name (Physics, Chemistry, Maths, Biology)

- **Marks Table**
  - Student ID
  - Subject-wise marks

## 📌 Key Features

- Organized relational dataset (3-table model)
- Subject-wise performance analysis
- Total and average score calculation
- Identification of top performers
- Pass/Fail analysis

## 📊 Analysis Performed

- Total marks per student
- Average marks calculation
- Subject-wise comparison
- Performance trends
- Ranking of students

## 🛠 Tools Used

- Microsoft Excel
  - Pivot Tables
  - Charts
  - Formulas (SUM, AVERAGE, IF)
